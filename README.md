# Mobile POS API

A backend API for a Mobile Point of Sale (POS) system. This API helps manage products, inventory, and sales transactions for retail applications.

## Features
- Product management (Create, Read, Update, Delete)
- Sales transaction handling
- Inventory management
- RESTful API architecture
- Easy integration with mobile POS apps

## Technologies Used
- Node.js
- Express.js
- REST API
- JSON
- Database (MySQL / MongoDB)

## Installation

1. Clone the repository

git clone https://github.com/rasadimanodya01/mobile_pos_API.git

2. Navigate to the project folder

cd mobile_pos_API

3. Install dependencies

npm install

4. Run the server

npm start

## API Endpoints

### Products

GET /api/products  
Get all products

GET /api/products/:id  
Get product by ID

POST /api/products  
Create new product

PUT /api/products/:id  
Update product

DELETE /api/products/:id  
Delete product
